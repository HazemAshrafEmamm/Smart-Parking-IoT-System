# 🚗 Smart Parking System - IoT Project

> **📢 Note:** For full technical details, hardware pictures, circuit diagrams, and the project demonstration video, please check the attached `Project_Report.pdf` and the video file in this repository.

## 📋 Project Overview
This project is an IoT-based Smart Parking system designed to provide real-time information about parking space availability.It integrates hardware sensors with a cloud-based Firebase backend and a web dashboard to help drivers easily locate empty spots and allow admins to monitor the facility.

## 🛠️ Hardware Components
* ESP32-DevKitC V4 Microcontroller.
* MFRC522 RFID Reader (for User Authentication).
* SG90 Servo Motor (acts as the Parking Gate).
* 5x Infrared (IR) Obstacle Avoidance Sensors.
* 10x LEDs (5 Green, 5 Red) for slot status.
* 16x2 I2C LCD Display.

## 💻 Software & Technologies
* **Embedded:** C++ (Arduino IDE).
* **Cloud Backend:** Google Firebase (Realtime Database, Authentication, Hosting).
* **Frontend Dashboard:** HTML, CSS, JavaScript.
